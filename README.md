# Aditi Baskaran Portfolio

A modern and responsive personal portfolio website built to showcase my projects, technical skills, achievements, internship experience, and interests in Full Stack Development, Machine Learning, Blockchain, Cloud Computing, and UI/UX Design.

The portfolio is designed with a premium modern UI featuring smooth animations, dark/light theme toggle, glassmorphism effects, responsive layouts, and interactive user experience.

## Features

* Modern responsive UI
* Dark & Light mode toggle
* Smooth animations and transitions
* Interactive project showcase
* Skills and coding profile section
* Achievement and experience timeline
* Contact section with social links
* Fully optimized for desktop and mobile devices

## Tech Stack

* React.js
* Tailwind CSS
* Framer Motion
* JavaScript
* Vite

## Sections Included

* Home
* About Me
* Skills
* Projects
* Experience
* Achievements
* Contact

## Getting Started

Clone the repository:

```bash
git clone https://github.com/aditibaskaran3/aditi_baskaran_portfolio.git
```

Navigate to project folder:

```bash
cd aditi_baskaran_portfolio
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
